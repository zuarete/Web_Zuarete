Tags de Texto

del

Representa uma parte do texto que foi excluída de um documento. Este elemento é
(não necessariamente) renderizado pelos navegadores com uma linha entre o texto.

Exemplos:

Esse texto será deletado

Esse texto não será exibido

Ele não pode ou não ser renderizado

ins

A tag define um texto que foi inserido em um documento. Os navegadores irão
geralmente sublinham o texto inserido.

Exemplos:

Vamos substituir essa palavra por essa.

Faltou complenttar com o argumento.

Eu gosto de fazer caminhada academia!

sub

A tag define o texto subscrito. O texto subscrito aparece meio caractere abaixo do
linha normal, e às vezes é renderizada em uma fonte menor. O texto subscrito pode
ser usado para fórmulas químicas

Exemplos:

Eu tomo H2O.

Agora devemos falar baixo, deste modo.

C6H6 Nunca consigo decorar isso.

sup

A tag define texto sobrescrito. O texto sobrescrito aparece meio caractere acima do linha
normal, e às vezes é renderizada em uma fonte menor. O texto sobrescrito pode ser usado
para notas de rodapé

Exemplos:

Isso é baseado nesse artigo1.

Não sei quanto é 223523.

Vamos usar essas referencias livro,1 video2 e a imagem3.

blockquote

A tag especifica uma seção que é citada de outra fonte.Os navegadores geralmente recuam
elementos, assim podendo fornecer informação de links ao usuario.

Exemplos:

Texto retirado do Wikipedia :

O palladianismo ou arquitetura palladiana é um estilo arquitetônico derivado da obra prática
e teórica do arquiteto italiano Andrea Palladio (1508-1580), um dos mais influentes personagens
de toda a história da arquitetura do Ocidente.

Texto retirado do G1 :

Os consumidores brasileiros gastaram R$ 6,42 bilhões em um total de pouco mais de 210 milhões
de encomendas internacionais em 2023, segundo a Receita Federal. A maior parte dos pacotes não
sofreu tributação, isto é, não precisou pagar impostos, taxas ou contribuições.

Texto retirado de História de Stars Insider :

Vamos ser honestos: todos temos um pouco de esquisitice em nós. Mas alguns de nós têm muito
mais do que outros, o que, às vezes, pode dificultar a vida cotidiana. Se você vive constantemente
com medo de estragar as coisas ou se preocupa com o que as outras pessoas pensam sobre você, então
você é definitivamente uma pessoa estranha. Quer provas?


q

A tag define uma citação curta. Os navegadores normalmente inserem aspas
ao redor da citação.

Exemplos:

Ela olhou para mim e falou : Até logo!

Uma vez perguntaram ao Dalailama - O que mais te surprende na humaninda ?

I have a dream ... Essa é uma famosa frase.

abbr

A tag define uma abreviação ou um acrônimo, como "HTML", "CSS", "O sr."

Exemplos:

Isso aconteceu na WW2.

O HTML é ultilizado para construir sites.

Quando queremos estilizar nosso site usamos CSS.

address

A tag define as informações de contato do autor/proprietário de um documento
ou um artigo. As informações de contato podem ser um endereço de e-mail, URL
endereço físico, telefone número, identificador de mídia social, etc. O texto no
elemento geralmente é renderizado em itálico, e os navegadores Sempre adicione
uma quebra de linha antes e depois do elemento.

Exemplos:

Isso aconteceu na WW2.

O HTML é ultilizado para construir sites.

Quando queremos estilizar nosso site usamos CSS.

cite

A tag define o título de um trabalho criativo (por exemplo, um livro, um poema, uma
música, um filme, uma pintura, uma escultura, etc.). O texto no elemento geralmente
é renderizado em itálico.

Exemplos:

Eu estou lendo Anos de chumbo do Chico Buarque.

Sou apaixonado pelo quadro A casa amarela do Van Gogh.

Uma das maiores expreções da escrita é Ulysses de James Joyce.

bdo

BDO significa Bi-Directional Override (Substituição Bidirecional). A tag é
usada para substituir a direção do texto atual. Em resumo ela vira o texto.

Exemplos:

Esse texto está ao contrario.

Só temos esse opção !

Até que é legal fazer isso !

code

A tag é usada para definir um pedaço de código de computador. O conteúdo dentro
é exibido na fonte monospace padrão do navegador.

Exemplos:

No Python temos a função len() para ler o tamanho de uma variavel.

Em HTML code é usado para mostrar um trecho de codigo.

Além dos outros comando como p, font, hr ...

pre

A tag define o texto pré-formatado. O texto em um elemento é exibido em uma
fonte de largura fixa e O texto preserva espaços e quebras de linha. O texto
será exibido exatamente como escrito no código-fonte HTML.

Exemplos: 

Esse   texto     tem   mais    espaços.

A sua fonte também é alterada.

Também  
                podemos     quebrar 
                   linhas.



kbd

A tag é usada para definir a entrada do teclado. O conteúdo dentro é exibido
na fonte monospace padrão do navegador.

Exemplos:

O texto que é exibido é do padrão Windons.

O comando para abrir o cmd é R + windons

Para acessar a barra de pesquisa aperte S + windons
