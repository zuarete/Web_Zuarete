Desenvolvimento Web

Tecnologias ultilizadas

Front-End

É a parte do site que o usuario ve, o programador responsavel
deve ter um bom senso estetico para juntar programação
com armonia dos elementospor
essa área ultiliza de meios como HTML e CSS para estilizar o site.

Back-End

É a parte responsavel pela logica e a funcionalidade dentro
do programa, como fazer calculos, atribuir função aos botões entre outros
assim para executar essas tarfas normalmente é ultilizado o JavaScript.

FullSatck

É a junção das funções de Back-End e Front-End, normalmente essa
função é encontrada em pequenas empresas, assim o profissional tem
que ter dominio de muias linguagens de programação.
